State Diff Viewer

Instantly see what changed in your state with visual highlighting.

How It Works

When an action is processed and the state changes, the diff viewer highlights:

  • Green - Properties that were added
  • Red - Properties that were removed
  • Yellow - Properties that were modified

Using the Diff View

  1. Select an action from the action log
  2. Click on the "Diff" tab in the state panel
  3. View the visual diff between the previous and current state
Note: The diff viewer only shows modified portions of the state, making it easy to focus on what changed.
