Quick Start

Get NgRx DevTool running in under 5 minutes.

1

Run the DevTool server

# From the ngrx-devtool directory
node dist/index.js
2

Open the DevTool UI

Navigate to http://localhost:3000

3

Start your Angular app

ng serve
4

View actions in the DevTool

All actions will appear with visual indicators:

User action
Effect result (shows which effect emitted the action)
