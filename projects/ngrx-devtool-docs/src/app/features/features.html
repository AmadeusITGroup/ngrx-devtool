Features

NgRx DevTool provides comprehensive tools for debugging and visualizing your Angular application's state management.

Real-time Monitoring

Track all dispatched actions as they happen in your application with live updates.

Effect Lifecycle

Monitor effect execution with start, emit, complete, and error state tracking.

Action Correlation

See which effect emitted each action with visual color-coded indicators.

Performance Tracking

Monitor reducer execution time, render timing, and state change performance.

State Visualization

Explore your application state with an interactive tree view.

Diff Viewer

Compare state changes between actions with highlighted differences.
